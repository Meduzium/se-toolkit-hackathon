┌─ USER FLOW ────────────────────────────────────────────────────────────────────┐
│                                                                                │
│  1. User sends song name to Telegram Bot                                    │
│     ↓                                                                        │
│  2. Bot calls /api/search endpoint                                          │
│     ↓                                                                        │
│  3. Backend uses yt-dlp to search YouTube                                   │
│     ↓                                                                        │
│  4. Results displayed as inline buttons (up to 10)                          │
│     ↓                                                                        │
│  5. User clicks "Download" on track                                         │
│     ↓                                                                        │
│  6. Bot calls /api/download endpoint                                        │
│     ↓                                                                        │
│  7. yt-dlp extracts MP3 from YouTube with ffmpeg                           │
│     ↓                                                                        │
│  8. MP3 sent to user via Telegram                                           │
│     ↓                                                                        │
│  9. Bot calls /api/log-search to record in database                         │
│     ↓                                                                        │
│  10. /top command aggregates all searches into charts                       │
│      Shows: "@john_doe - Bohemian Rhapsody by Queen (3)"                   │
│                                                                              │
└────────────────────────────────────────────────────────────────────────────────┘

│  📦 LOCAL SETUP (5 minutes)                                                  │
│                                                                              │
│  1. Install dependencies:                                                   │
│     cd backend && pip install -r requirements.txt                           │
│     cd ../bot && pip install -r requirements.txt                            │
│                                                                              │
│  2. Configure environment:                                                  │
│     Edit backend/.env: DATABASE_URL                                        │
│     Edit bot/.env: TELEGRAM_BOT_TOKEN                                      │
│                                                                              │
│  3. Create database:                                                        │
│     createdb music_bot_db                                                   │
│                                                                              │
│  4. Run services:                                                           │
│     Terminal 1: cd backend && uvicorn app.main:app --reload               │
│     Terminal 2: cd bot && python main.py                                   │
│                                                                              │
│  🐳 DOCKER SETUP (2 minutes)                                                │
│                                                                              │
│  export TELEGRAM_BOT_TOKEN=your_token_here                                 │
│  docker-compose up --build                                                 │
