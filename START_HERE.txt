🚀 QUICK START:

1️⃣  VERIFY PROJECT:
    cd music_bot_project
    python verify_project.py

2️⃣  SETUP DATABASE:
    createdb music_bot_db
    # or
    psql -U postgres -c "CREATE DATABASE music_bot_db;"

3️⃣  CONFIGURE ENVIRONMENT:
    Edit backend/.env:
      DATABASE_URL=postgresql://user:password@localhost:5432/music_bot_db
      GENIUS_API_KEY=your_key_here  (optional)
    
    Edit bot/.env:
      TELEGRAM_BOT_TOKEN=your_token_here
      BACKEND_URL=http://localhost:8000

4️⃣  INSTALL DEPENDENCIES:
    cd backend
    python -m venv venv
    venv\Scripts\activate  # Windows
    source venv/bin/activate  # macOS/Linux
    pip install -r requirements.txt
    
    cd ../bot
    python -m venv venv
    venv\Scripts\activate  # Windows
    source venv/bin/activate  # macOS/Linux
    pip install -r requirements.txt

5️⃣  RUN SERVICES:
    Terminal 1 (Backend):
      cd backend
      venv\Scripts\activate
      uvicorn app.main:app --reload
    
    Terminal 2 (Bot):
      cd bot
      venv\Scripts\activate
      python main.py

6️⃣  TEST ON TELEGRAM:
    - Search for your bot on Telegram
    - Send /start
    - Search for a song
    - Use /top to see trending tracks

❓ FREQUENTLY ASKED QUESTIONS:

Q: Do I need Docker?
A: No, you can run locally with Python virtual environments.
   Docker is optional but recommended for production.

Q: Do I need ffmpeg?
A: Yes, for MP3 extraction. Install via:
   Windows: choco install ffmpeg
   macOS: brew install ffmpeg
   Linux: apt-get install ffmpeg

Q: Do I need a Genius API key?
A: No, it's optional. Only needed for album art display.

Q: How do I get a Telegram bot token?
A: Search @BotFather on Telegram, send /newbot and follow prompts.

Q: What's the default database password?
A: Default is 'password'. Change in .env and docker-compose.yml.
